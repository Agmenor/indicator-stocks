INSTALL

Icons:
format svg
place in /usr/share/icons/hicolor/scalable/status/
execute sudo gtk-update-icon-cache /usr/share/icons/hicolor/

Help:
copy help directory content to /usr/share/gnome/help/indicator-stocks/

init debian file for packaging:
dh_make -c gpl3 -e ce3a@gmx.de -p indicator-stocks_0.1.0 -f indicator-stocks.tar.gz
